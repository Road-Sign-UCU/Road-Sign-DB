database_convertion.AnalysisADT module¶

Contains an ADT for data analysis Requirements for the SignPointArray ADT: 1) from_file(filename) -- read the data from the file. 2) to_file(filename) -- write the data to the file. 3) add_entry() -- add an entry to the Array. Uses named parameters. 4) analyse() -- get all the needed statistics (may be split into multiple methods)
